ℹ️Tentang Aplikasi Analisis Sentimen

Apa ini?

Aplikasi ini adalah dashboard analisis sentimen yang menampilkan visualisasi, wordcloud, dan tabel data dari dataset sentimen. Website ini dibuat untuk memudahkan eksplorasi dan pemahaman data sentimen secara interaktif.

Sumber Data

Data yang digunakan berasal dari file data/data_sentimen.csv yang berisi kolom text dan sentiment.

Cara Menggunakan

  • Dashboard: Melihat ringkasan statistik dan insight otomatis.
  • Visualisasi: Melihat grafik distribusi sentimen secara interaktif.
  • WordCloud: Melihat wordcloud kata-kata yang sering muncul per sentimen.
  • Tabel Data: Melihat dan mencari data mentah, serta download CSV.

Teknologi

Website ini dibuat dengan Python Flask, Bootstrap, Chart.js, dan DataTables.js.
